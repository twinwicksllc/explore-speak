# Explore Speak

An RPG-style language learning app powered by AI-driven guides that helps users learn languages through immersive quests and proven spaced repetition techniques.

## Project Overview

Explore Speak combines gamification with research-backed language learning pedagogy to create an engaging and effective learning experience. Users embark on quests with AI guides to learn vocabulary, grammar, and cultural context.

## Current Status

**Phase 1 MVP - Sprint 1: Production-Ready Foundation** (In Progress)

## Technology Stack

- **Frontend:** React 18 + Vite
- **Backend:** AWS Lambda (Node.js 20.x)
- **Database:** Amazon DynamoDB
- **Authentication:** AWS Cognito
- **Audio:** AWS Polly
- **CDN:** Amazon CloudFront
